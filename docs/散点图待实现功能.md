## 渐隐退场动画

- **触发时机**：事件掉出榜单时（新数据到达 → diff 发现某事件消失）
- **表现**：在该事件最后一个圆点位置放置 `effectScatter`，做淡出/向右消散动效
- **实现要点**：
  - JS 侧维护上一次数据快照（`prevData`）
  - 每次轮询拿到新数据后做 diff：`prevKeys - newKeys` = 掉出的事件
  - 对每个掉出事件，取其最后一个 data point，创建临时 effectScatter series
  - 动画结束后（约 1-2s）移除该临时 series
- **依赖**：悬停连线 + NEW 脉冲稳定后再做
